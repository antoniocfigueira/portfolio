## Sobre o Projeto
Este projeto consiste no meu portfólio pessoal, desenvolvido com o objetivo de apresentar:

- informação sobre mim  
- formação académica  
- projetos realizados  
- contactos profissionais  

O site utiliza HTML, CSS para estilização e JavaScript para interatividade.

## Funcionalidades
- layout responsivo  
- menu de navegação adaptado a mobile  
- fotografia e apresentação pessoal  
- lista cronológica de formação  
- grelha de projetos
- formulário de contacto com validação HTML nativa
- mensagem de sucesso após envio do formulário  
- mapa incorporado com iframe  
- links para GitHub, LinkedIn, email e telefone

## Acessibilidade
- imagens com texto alternativo  
- labels associadas a inputs  
- contraste adequado de cores
